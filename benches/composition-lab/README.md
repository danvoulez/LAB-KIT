# composition-lab

This bench is an operational study surface, not a placeholder.

It must run against LogLine Acts from `ops.logline_acts`, use projections only as read models, record any experiment as evidence, and leave a ghost when a claim cannot be closed.

Minimum bench loop:

1. declare the question as a LogLine Act;
2. run the experiment or inspection;
3. store evidence references;
4. prepare a receipt candidate only for the narrow claim proven;
5. keep unresolved conditions as ghosts.
